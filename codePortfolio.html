This is the code I've made during the 2017 Girls Who Code Summer Immersion Program.

This is a parallel circuit using two LED lights and one button. One light is already lit cause it is a closed loop while the other isn't since it is open due to the button, so if you press it...

It lights up the other one! And of course if you let go it will turn it off while the other one still lights up.
